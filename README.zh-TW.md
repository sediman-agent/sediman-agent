### 安裝

```
curl -fsSL https://raw.githubusercontent.com/sediman-agent/OpenSkynet/main/install | bash
```

設定 API 金鑰後即可使用：

```
export OPENAI_API_KEY=sk-...
openskynet
```

### 智慧代理

按 `Tab` 鍵在智慧代理之間切換。

- **Manager（管理者）** — 編排任務，調度其他智慧代理
- **Browser（瀏覽器）** — 瀏覽網頁、填寫表單、點擊按鈕
- **Coder（編碼者）** — 撰寫、編輯和除錯程式碼
- **Terminator（終結者）** — 自主執行，無需確認

### 功能特色

- **示範即學習** — 看你操作一次瀏覽器，隨時自動回放
- **自我修復** — 網頁變了？自動適配修復
- **7×24 排程** — 基於 cron 的定時自動化，睡覺也在跑
- **技能中心** — 瀏覽和安裝社群技能
- **持久記憶** — 跨工作階段記住你的偏好
