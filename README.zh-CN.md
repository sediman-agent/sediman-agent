### 安装

```
curl -fsSL https://raw.githubusercontent.com/sediman-agent/OpenSkynet/main/install | bash
```

设置 API 密钥后即可使用：

```
export OPENAI_API_KEY=sk-...
openskynet
```

### 智能体

按 `Tab` 键在智能体之间切换。

- **Manager（管理者）** — 编排任务，调度其他智能体
- **Browser（浏览器）** — 浏览网页、填写表单、点击按钮
- **Coder（编码者）** — 编写、编辑和调试代码
- **Terminator（终结者）** — 自主执行，无需确认

### 功能特性

- **示范即学习** — 看你操作一次浏览器，随时自动回放
- **自我修复** — 页面变了？自动适配修复
- **7×24 调度** — 基于 cron 的定时自动化，睡觉也在跑
- **技能中心** — 浏览和安装社区技能
- **持久记忆** — 跨会话记住你的偏好
